## Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. For Jekyll integration:
   ```bash
   bundle install
   bundle exec jekyll serve
   ```

## Deployment

This site is designed as a static website and can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- AWS S3 + CloudFront
- Any static hosting provider
